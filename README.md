## Technologies Used

- React Native
- Expo

To test the project all you need to do is to clone the repository:

```sh
git clone https://github.com/pedroelton/Instagram-clone-React-Native.git
```

Enter in the file:

```sh
cd Instagram-clone-React-Native
```

Install all the dependencies:

```sh
npm install
```

or

```sh
yarn
```

Start the project:

```sh
expo start
```
